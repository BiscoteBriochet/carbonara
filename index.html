How to Configure Your Train Network

HSR LinesLSR (Local Service Rail)

Regular train lines with sequential stations:

  • Define LSR lines with "type": "lsr"
  • Each segment connects exactly two stations with a specific travel time
  • Trains must travel through all intermediate stations

HSR (High Speed Rail)

Express lines with direct connections between multiple terminuses:

  • Define HSR lines with "type": "hsr"
  • List all terminuses in the "terminuses" array
  • Define direct connections between terminuses in the "connections" array
  • Each connection has "from", "to", and "time" properties
  • HSR allows direct travel without passing through other terminuses

Example Configuration

The system comes with example data showing both LSR and HSR networks. Click the "Configuration" tab to see and edit it.
